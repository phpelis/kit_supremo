# 🚀 Kit Supremo — The Ultimate AI Development Kit

Bem-vindo ao **Kit Supremo**, um ambiente de desenvolvimento de elite projetado para máxima sinergia entre humanos e agentes de IA (Antigravity, Cursor, Claude Code).

Este repositório combina as metodologias de `superpowers`, `antigravity-workspace-template` e `antigravity-kit` em um sistema unificado de alta performance.

## 🧠 Arquitetura Orientada a Agentes (Agent-First)

O projeto é orquestrado através da pasta `.agents/`, que contém:
- **`skills/`**: Habilidades especializadas (n8n, Chatwoot, TDD, Design, etc.).
- **`methodology/`**: Fluxos de trabalho rigorosos (Brainstorm -> Plan -> TDD -> Verify).
- **`rules/`**: Regras mestre de engenharia ([MASTER.md](file:///c:/Users/phpel/OneDrive/Documentos/kit_supremo/.agents/rules/MASTER.md)).

## 🛠️ Tecnologias & Integrações Core

- **Backend/Scripts**: Node.js & Python (Engine em `.agents/engine/`).
- **Automação**: [n8n Master Class](file:///c:/Users/phpel/OneDrive/Documentos/kit_supremo/.agents/skills/integrations-n8n/SKILL.md).
- **Atendimento**: [Chatwoot Specialist](file:///c:/Users/phpel/OneDrive/Documentos/kit_supremo/.agents/skills/integrations-chatwoot/SKILL.md).
- **Frontend**: Vite/React (se aplicável ao projeto src).

## 🚀 Como Operar o Kit

### 1. Instruções para a IA
Este kit possui um "Strong Rule" de conformidade. Sempre que iniciar uma tarefa:
1. Comece pelo **Brainstorming** (`/brainstorm`).
2. Crie um **Plano de Implementação** (`/plan`).
3. Siga o **TDD Estrito** (`/tdd`).
4. Verifique antes de finalizar (`/verify`).

### 2. Transplante para Novo Projeto (Suprematize)

Se você estiver em um novo diretório e quiser transformar um projeto comum em um **Kit Supremo**:

**No Windows (PowerShell):**
```powershell
irm -uri "https://raw.githubusercontent.com/phpel/kit_supremo/main/suprematize.ps1" | iex
```

**No Linux/macOS (Bash):**
```bash
curl -sSL https://raw.githubusercontent.com/phpel/kit_supremo/main/suprematize.sh | bash
```

> [!CAUTION]
> Isso irá substituir arquivos `CLAUDE.md` e `.cursorrules` para garantir o alinhamento com o Kit Supremo.
